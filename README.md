# AI Chat Assistant (React)

## Project Overview
This is a simple AI Chat Application built using React.  
Users can type a message and receive a response (demo AI).

## Technologies Used
- React.js
- JavaScript (ES6)
- HTML & CSS

## Features
- User input handling
- Dynamic message display
- Chat interface (basic)
- AI response simulation

## Future Improvements
- Real AI API integration
- Chat history UI
- Backend (Node.js) integration

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
